deal.II & CUDA

Massive GPU Parallelization

04 / 2013

Role: Developer

For my 4th year Electrical Engineering thesis, I chose to optimize a finite element analysis library called deal.II by running its matrix operations over the hundreds of cores offered by modern GPUs. I did so by adapting existing C++ code within the library to CUDA, nVidia's parallel architecture.

Deal.ii is used heavily in academia to solve complex differential equations. Along with 2 partners, we were able to achieve a speedup of 1.5 to 2.5 times the original, sequential speed. This marks a significant improvement over the current sequential method. We are releasing our work under the open source "QPL" license; it will be included in the next release of the program.

The full report can be read below. tl;dr: massively parallel processors are an effective way to improve the performance of non-sequential systems.
